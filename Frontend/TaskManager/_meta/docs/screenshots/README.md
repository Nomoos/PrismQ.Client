## Screenshot Guidelines

- **Resolution**: 1080x1920 (mobile portrait) or 360x640 (CSS pixels)
- **Format**: PNG with compression
- **Device**: Preferably Redmi 24115RA8EG or similar
- **Quality**: High quality, clear text
- **Privacy**: Remove any sensitive data
