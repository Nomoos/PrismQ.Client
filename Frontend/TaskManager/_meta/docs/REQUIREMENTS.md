# Frontend/TaskManager - Requirements

**Last Updated**: 2025-11-11

## Development Requirements

- **Node.js**: 18+ or higher
- **Package Manager**: npm
- **OS**: Linux, macOS, or Windows

## Production Server Requirements

- **Web Server**: Apache with mod_rewrite enabled
- **PHP**: 7.4+ (for deployment scripts only)
- **Backend**: Backend/TaskManager API must be running and accessible
- **Hosting**: Compatible with shared hosting (Vedos/Wedos)

## Browser Requirements

### Desktop
- Chrome/Edge (latest 2 versions)
- Firefox (latest 2 versions)
- Safari (latest 2 versions)

### Mobile
- Chrome Android (latest 2 versions)
- Safari iOS (latest 2 versions)

## Network Requirements

- **Development**: Broadband internet for npm package downloads
- **Production**: 3G or better for optimal user experience

## Storage Requirements

- **Development**: ~500MB for node_modules
- **Production**: ~5MB for deployed application
