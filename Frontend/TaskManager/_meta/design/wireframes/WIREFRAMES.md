1. **Filter Tabs**: Tap to filter tasks by status

3. **Swipe Right** (Pending tasks): Quick claim action

5. **Pull Down**: Refresh task list

### States

- **Loading**: Skeleton cards while fetching
- **Empty**: "No {filter} tasks" message
- **Error**: Error banner with retry button

- **Valid**: Green checkmark, green border
- **Invalid**: Red error message, red border
- **Pristine**: Default gray border

- "This field is required"
- "Must be a valid number"
- "Must be at least 1"
- "Maximum 50 characters"

1. **Task Type Dropdown**: Select from available task types
2. **Parameters**: Dynamic based on selected task type
3. **Create Button**: Validate and submit
4. **Cancel Button**: Confirm discard changes if dirty

### Setting Types

**Toggle Settings**:
- Auto-claim tasks
- Dark mode
- Notifications
- Sound effects

**Text Input Settings**:
- Worker ID (read-only or editable)
- API endpoint (advanced)

**Number Input Settings**:
- Max concurrent tasks (1-10)
- Polling interval (10-120 seconds)

**Dropdown Settings**:
- Auto-refresh interval (10s, 30s, 60s, Never)
- Theme (Light, Dark, Auto)
- Language (English, Spanish, etc.)

## Responsive Behavior

### Mobile (360px-639px)
- Single column layout
- Bottom navigation
- Full-width cards
- Stack elements vertically

### Tablet (640px-1023px)
- 2-column grid for cards
- Bottom navigation or side tabs
- Wider cards with more content visible
- Some horizontal layouts

### Desktop (1024px+)
- Multi-column layouts
- Sidebar navigation (persistent)
- Grid layouts (2-3 columns)
- More whitespace
- Max-width containers (1280px)

### Color Legend for Dark Mode

**Backgrounds**:
- `#0d1117` - Main canvas (page background)
- `#161b22` - Surface (cards, modals, navigation)
- `#1c2128` - Elevated surface (hover states)

**Borders**:
- `#30363d` - Default borders
- `#21262d` - Muted borders
- `#6e7681` - Strong borders

**Text**:
- `#e6edf3` - Primary text (15.8:1 contrast)
- `#8d96a0` - Secondary text (7.2:1 contrast)
- `#6e7681` - Tertiary text (5.1:1 contrast)

**Status (Subtle Backgrounds)**:
- `#0d419d` - Info/Pending (blue)
- `#341a03` - Warning/Claimed (orange)
- `#0d3818` - Success/Completed (green)
- `#490b08` - Error/Failed (red)

**Status (Text)**:
- `#58a6ff` - Info/Primary actions
- `#d29922` - Warning
- `#3fb950` - Success
- `#ff7b72` - Error

## Accessibility Notes

**All Wireframes Include**:
- Focus indicators on all interactive elements
- Keyboard navigation support
- Screen reader compatible structure
- Touch targets minimum 44x44px
- Color contrast WCAG 2.1 AA compliant
- Loading states announce to screen readers
- Error messages associated with form fields

**Dark Mode Accessibility**:
- All text colors meet WCAG 2.1 AAA contrast ratios (7:1+)
- Primary text: 15.8:1 contrast on dark backgrounds
- Status colors: 7.1:1 to 8.6:1 contrast
- Focus indicators highly visible (#58a6ff)
- Reduced eye strain in low-light environments
