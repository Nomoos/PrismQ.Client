- **OS**: Android 14 (HyperOS)

### Design Constraints
- Touch targets: 44x44px minimum
- Font sizes: 16px minimum for body
- Portrait-primary orientation
- High contrast for outdoor visibility
- AMOLED-optimized (dark theme option)

## Design Principles

### Mobile-First
- Design for mobile (360px) first
- Scale up to tablet and desktop
- Progressive enhancement
- Touch-optimized interactions

### Accessibility
- WCAG 2.1 AA compliance
- Color contrast 4.5:1 minimum
- Screen reader compatible
- Keyboard navigable
- Focus indicators

### Performance
- Lightweight visuals
- System fonts preferred
- Optimized images
- Minimal animations
- Fast loading feedback

### Task-Focused
- Clear task status indicators
- Quick actions (claim, complete)
- Minimal steps to complete tasks
- Clear error states
- Progress visibility
