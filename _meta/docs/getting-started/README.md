## Prerequisites

Before you begin, ensure you have:
- Python 3.10 or higher
- Node.js 18 or higher (see [Node.js Installation](NODEJS_INSTALLATION.md))
- npm package manager

## Recommended Reading Order

For first-time users:
1. Start with [Node.js Installation](NODEJS_INSTALLATION.md) if you don't have Node.js
2. Follow the [Setup Guide](SETUP.md) for complete installation
3. Read the [User Guide](USER_GUIDE.md) to learn how to use the application
4. Keep [Troubleshooting](TROUBLESHOOTING.md) handy for common issues
